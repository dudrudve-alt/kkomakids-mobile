# 꼬마키즈

4-5세 자녀를 위한 한글·영어 학습 웹앱

## 기능

- 📚 한글 자모 카드 (자음·모음 24장)
- 🔤 영어 알파벳 카드 (A-Z 26장)
- 🍎 한글·영어 단어 카드
- ✏️ 글자 완성 게임 (10문제, 80% 통과)
- 👏 음절 박수
- 🎮 미니게임과 자동차 컬렉션

## PC/모바일 통합

- 하나의 GitHub Pages 링크로 PC, 태블릿, 모바일 지원
- 화면 크기에 맞춰 메뉴와 게임판 자동 조정
- iOS Safari / Android Chrome 모두 지원
- 진동 피드백
- safe-area (노치/홈바 영역 대응)
