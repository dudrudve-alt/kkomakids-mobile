# 꼬마키즈 모바일

4-5세 자녀를 위한 한글·영어 학습 웹앱 (모바일 전용)

## 기능

- 📚 한글 자모 카드 (자음·모음 24장)
- 🔤 영어 알파벳 카드 (A-Z 26장)
- ✏️ 글자 완성 게임 (10문제, 80% 통과)

## 모바일 특화

- 반응형 디자인 (폰 + 태블릿)
- iOS Safari / Android Chrome 모두 지원
- 진동 피드백
- safe-area (노치/홈바 영역 대응)
- 더블탭 줌 차단
